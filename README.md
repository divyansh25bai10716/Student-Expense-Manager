#  Student Expense Manager

A simple Python-based console application designed to help college students track income and expenses.  
It stores data locally and allows students to monitor their spending habits and remaining balance.

---

##  Overview

College students often receive pocket money from parents, scholarships, or part-time work — but many struggle to track where the money goes.  
This project provides an easy solution to manage finances without complicated apps.

---

##  Features

-  Add income (Parents, scholarship, job, etc.)
-  Record expenses (Food, travel, books, etc.)
-  View available balance
-  See complete transaction history
-  Data automatically stored in a JSON file
-  Persistent data — continues from last use
